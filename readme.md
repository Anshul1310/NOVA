NOVA - Next-gen optical video accelerator

An Android camera application built using the Camera2 API that records high frame rate video (120/240 FPS), provides camera controls, and saves metadata alongside recorded videos.
This project demonstrates understanding of high-speed camera pipelines, Camera2 constrained high-speed sessions, and manual camera controls.
- Uses Constrained high speed capture sessions
- Exposure compensation supported
- Allows to modify ISO and shutter value even in high frame rates
- Records high frame rate videos
- Allows to share aforementioned video

Technologies/Tools used;
- JAVA
- Android Camera2 API
- Shared preferences for local storage
- Adobe Illustrator
- FIGMA
- Lotti Files
- Github
